Example Players

In the distribution binary, the c++_example and java_example directories contain sample players implemented in C++ and Java. The source code for these demonstrates how a player is to interact with the game, and they may give you some ideas about how your own player could operate. Of course, these players are not too smart. Using them as a starting point, you can probably do much better.

You can compile these players independently and then run them against each other in the game using a command like the following. If you've compiled the example players, and left their executables in the same directories as the source code, you should be able to run this command from the top-level directory of the expanded game distribution binary : 

java -jar coderunner.jar -player cpp c++_example/runAndDig -player java -cp java_example GoToNearestGoldPlayer

Source code for the Java player is stored in a java_example subdirectory, but its code is defined in the default package. The -cp java_example on the command line adds java_example to the class path before running this player. If you put your Java player in the current directory, you won't need the -cp option.

C++ Example Player

For each turn, the C++ sample player, runAndDig.cpp first determines which directions it can move. Then, it randomly chooses a direction, preferring to move to cells that it hasn't visited recently. As it moves, the player turns away from nearby enemies and creates a hole for the enemy to fall into if there's an opportunity.

  • runAndDig.cpp : source code for the cpp example player.

Java Example Player

The Java sample player, GoToNearestGoldPlayer, simply calculates distances to all cells on the map containing Gold. It chooses the nearest one and goes toward it. For calculating distances and choosing movement direction, this player uses a breadth-first search of the map, based on the rules for player movement. This player never tries to destroy bricks, and it doesn't try to avoid enemies.

  • GoToNearestGoldPlayer.java : Main file for the Java player.
